# my-react-boilerplate

### What's on board

- react
- react-router
- lodash
- **styles:** SASS (scss), autoprefixer
- **bundler:** webpack
- **test stack:** karma, mocha, chai, sinon

#### Development mode:

``` bash
  $ npm start
```  

#### build:

``` bash
  $ npm build
```  

#### build in production mode:

``` bash
  $ npm release
```
